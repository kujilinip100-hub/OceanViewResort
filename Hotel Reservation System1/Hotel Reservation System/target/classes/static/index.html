    Help & Documentation

    How to add a new reservation?

    Navigate to 'Add Reservation' in the sidebar, fill in guest details, select room type and dates, then click 'Confirm Booking'. Reservation ID will be generated automatically.

    How is the bill calculated?

    Bill = (Check-out Date - Check-in Date) x Price Per Night. Price: Standard ($50), Deluxe ($80), Suite ($120).

    Retrieving booking records?

    Use the 'Booking Records' page to list all guests or search by Reservation ID to find a specific entry.
